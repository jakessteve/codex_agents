# Entity Types

status: scaffold
owner: librarian
summary: Define canonical node types for project docs, agents, contracts, tests, and runtime events.
## Entity Types

| Type | Use |
|---|---|
| project | Root project identity. |
| document | Durable artifact. |
| requirement | Business, product, or technical requirement. |
| epic | Delivery slice. |
| story | User or implementation story. |
| task | Atomic execution item. |
| agent | Role that worked on the item. |
| contract | Task or handoff contract. |
| test | Validation action or suite. |
| evidence | Proof of execution. |
| decision | Routing or architecture decision. |
| trace | Observable execution record. |
