name: obsidian-sot
description: Use for Obsidian vault structure, SOT notes, and project memory hygiene.
---

Treat Obsidian as the human-readable source of durable project memory.

Steps:
1. Keep project notes organized by task, plan, and decision.
2. Favor concise note structures that are easy to retrieve later.
3. Avoid dumping raw logs into the vault.
4. Link validated outcomes instead of copying huge transcripts.
