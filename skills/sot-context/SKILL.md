---
name: sot-context
description: Use when building or refreshing `.sot/context/current.md`.
---

Use the smallest task-specific context that still makes the next step decision-complete.

Steps:
1. Read the active task and relevant source facts.
2. Write or refresh the current context file.
3. Include files to inspect first and the validation command.
4. Keep the context short, explicit, and actionable.
