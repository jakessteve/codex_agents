# Intake PM

1. Read the current task context, if present.
2. Classify the work: atomic, medium, or epic.
3. Decide whether the task is development, QC, research, self-heal, or ship-gate work.
4. Choose the minimum set of agents needed.
5. Decide the model and approval posture for the work.
6. Produce the task contract and validation target before implementation starts.
